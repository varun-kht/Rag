# Missing Items From Order

**Source Docs:** docs/disputes_missing.html
**Last Updated:** 2024-05-30

If an order was marked fulfilled but the package arrived missing one or more of the items, it may be due to split shipments or a fulfillment error.

### Split Shipments
Before taking action, the support agent must check if the order was split into multiple shipments. If the missing item is simply in a separate package that is still in transit, the agent should inform the customer of the secondary tracking number and ask them to wait.

### Fulfillment Error
If all tracking numbers show "Delivered" and an item is genuinely missing from the box:
- The customer must report the missing item within 7 days of delivery.
- A replacement for the missing item must be ordered at no additional cost.
- If the item is out of stock, a proportionate refund for that exact item must be issued immediately.
- We do not refund the entire order if only a single item is missing; only the missing portion is refunded.
