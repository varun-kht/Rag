# General Return Policy

**Source Docs:** docs/returns_general.html
**Last Updated:** 2024-05-10
**Applicability:** First-Party Items Only

We want you to be completely satisfied with your purchase from ABC. 
Our general return policy allows customers to return most new, unopened items sold and fulfilled by ABC within 30 days of delivery for a full refund.

### Return Window
Items must be returned within 30 days from the date of the `delivery_date` marked on the order. If the 30-day window has passed, the item is generally ineligible for a return unless covered by a specific exception.

### Condition of Returned Items
To be eligible for a full refund, items must be in new condition, with original packaging and all accessories included. 
Opened items that are not defective may be subject to a restocking fee of up to 15% of the item's purchase price, at the discretion of the support agent or receiving facility.

### Return Shipping Costs
If the return is due to our error (e.g., incorrect item sent, defective item), ABC will cover the return shipping costs. Hand out a prepaid return label.
If the return is discretionary (e.g., "changed my mind", "no longer needed"), the cost of the return shipping label will be deducted from the customer's final refund amount. Standard return shipping deduction is $5.99.

### Initiation
Customers can initiate a return through their online account portal or by contacting customer support. An RMA (Return Merchandise Authorization) number will be generated for all valid returns.
