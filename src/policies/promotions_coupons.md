# Promotions and Coupons

**Source Docs:** docs/promotions_coupons.html
**Last Updated:** 2023-11-20

ABC frequently runs promotional campaigns including promo codes, sitewide sales, and discounts.

### Applying Coupons
Coupons must be applied at the time of checkout. If a customer forgot to apply a valid, unexpired coupon code during checkout, a support agent can apply it retroactively as a partial refund, provided the contact is within 7 days of the order date.
If the contact is past 7 days, the coupon cannot be applied retroactively.

### Stacking Policies
Customers may use only **one** promotional code per order. Coupons cannot be stacked or combined with other offers, unless expressly stated by the terms of the specific promotion.
If an item is already on a sitewide sale, coupon codes cannot be applied to that item to further reduce the price.

### Expiry and Single-Use
Each unique promotional code can only be used once per customer account. Expired coupons cannot be re-activated, and support agents must not provide a replacement code for an expired offer.
